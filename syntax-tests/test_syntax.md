# Markdown Syntax Highlighting Demo

This is a **demonstration** of *syntax highlighting* in **Markdown**.

## Features

### Code Blocks

Inline code: `let x = 42;`

```rust
fn main() {
    println!("Hello, world!");
}
```

### Formatting

- **Bold text** with double asterisks
- *Italic text* with single asterisks
- __Bold with underscores__
- _Italic with underscores_

### Links

Check out [GitHub](https://github.com) or [Rust](https://rust-lang.org).

### Headings

# H1 Heading
## H2 Heading
### H3 Heading
#### H4 Heading

### Lists

1. First item
2. Second item
3. Third item

- Bullet point
- Another bullet
- Last one

### Mixed Content

Here's some **bold** text with *italic* mixed in, plus `code` and a [link](https://example.com).

You can also combine **_bold and italic_** together.

### Code Examples

```python
def hello(name):
    return f"Hello, {name}!"
```

```javascript
const greeting = (name) => {
    return `Hello, ${name}!`;
};
```
